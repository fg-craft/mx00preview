# mx00preview

## What is mx00?
mx00 is a prototype VSTi currently in development, utilizing common Phase Modulation. 

It uses the JUCE framework.
- Up to 8 oscillators available per note.
- Oscillators can use arbitrary waveforms per cycle, plus Soundfont and external input (planned).
- Multi-stage Envelope Generator, LFO, Keyboard Scaling, Modulation Matrix, FILTER, PAN, and FX available per oscillator (planned).
- The modulation matrix will allow specifying PM, FM, AM, and RM per oscillator (planned).

## Progress, or Rather, My Motivation
- The PM engine is complete.
- Oscillator settings are now available.
- PAN settings are now available.
- Modulation Matrix is now available.
- Multi-stage Envelope Generator is now available.
- Implementing the GUI is overwhelmingly more difficult than implementing the modulation engine.
